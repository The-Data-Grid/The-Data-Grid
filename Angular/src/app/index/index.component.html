The Data Grid is a free and open source database creation, data collection, and data management platform for environmental auditing.
Open data for everyone
The Data Grid hosts open databases in the cloud so anyone can view, filter, sort, and download publicly available data easily and at no cost.

The platform allows organizations to upload their data to their database, manage auditors, and view the data they have uploaded. This way, environmental data can be easily standardized within and across organizations.

How it works
The Data Grid allows you to generate a custom PostgreSQL database without code, and manage that database in the cloud using thedatagrid.org. Databases schemas are created using a universal data representation format that we invented in order to allow people to make databases without needing to know how they work internally. With this:
  • Generate preformatted spreadsheets for data upload <
  • Manage data manipulation privileges <
  • Perform complex queries on the data <
