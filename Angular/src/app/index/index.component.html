Summary of Services

Data Collection Infrastructure

Upload audits using our iOS app or through our web form. The iOS app supports offline auditing. Crew Leaders can manage audits through the Audit session Management Dashboard.

Data Filtering and Management

View, filter, sort, and edit audit data in a spreadsheet-style format on the Data page.

Automated Data Analysis

Run analysis to identify target areas that yield the highest environmental and financial benefits when improved.

Fundraising Platform

Easily post a Facility Improvement project on the TDG Sustainability Project Platform and crowdfund improvements.

Financial -> reduce inefficient features
